# SSE Project

This project is a full-stack application built with the following technologies:

## Backend

- **Express.js (TypeScript):** A web application framework for Node.js used to build the API.
- **PostgreSQL:** An open-source relational database management system.
- **Prisma ORM:** A next-generation ORM for Node.js and TypeScript to interact with the database.

## Frontend

- **React (TypeScript):** A JavaScript library for building user interfaces.
- **Vite:** A build tool that aims to provide a faster and leaner development experience for modern web projects.

## Infrastructure

- **Docker:** Used for containerizing the PostgreSQL database.
